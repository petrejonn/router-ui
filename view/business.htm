This tab allows you to switch perormance of your smart network router either for personal mode or buisness mode

Personal :

This is for general use case that require no voucher code, no data limits on connected devices. it is preferred for personal and home use. Benefits of Personal Mode Easy Device Connection as this doesnt require voucher codes for login for everyday personal use as users can track their data cosumptions on multiple devices yet with no data limits, splash screen page is disabled as user need not to login

Buisness:

This option is for buisness use case where device admin makes profit from connected devices by resellling data using voucher codes and data limits. This prefered for buisness use, official use and public domain use with restrictions to voucher access. Benefits of Buisness Mode allows users to earn based on connected device splash screen page is enable to allow only users with voucher codes login this is a security feature allocate different data limits to voucher codes
